# Changelog

All notable changes will be documented in this file.

## [0.1.0] - 2026-05-19

### Added

- Initial release
- `analyze` command - Analyze business websites for operational inefficiencies
- `outreach` command - Generate personalized outreach messages
- `config` command - Manage configuration (API key storage)
- Operational signal detection (WhatsApp booking, manual scheduling, etc.)
- AI-powered analysis using OpenAI GPT-4o
- Beautiful terminal UI with chalk and boxen
- Loading spinners with ora
- Support for npx, bunx, pnpm, yarn

### Features

- Website scraping with Playwright
- JSON output option (`--json` flag)
- Configurable API key (env var or stored config)
- Cross-platform package managers support
